## 🚀 **How to Use**

### Quick Start
```bash
# Install dependencies
pip install -r requirements.txt

# Analyze a GIA trading statement
python3 gia_trading_analyzer.py path/to/trading_statement.pdf

# Calculate P&L from extracted transactions
python3 gia_pnl_calculator.py
```

### For Other PDF Formats
```bash
# Try generic analyzer for non-GIA PDFs
python3 simple_pdf_analyzer.py path/to/trading_statement.pdf
```
